# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-29

### Added

- Complete Finite Automata engine supporting DFA, NFA, and ε-NFA.
- Visual workspace with interactive design panel for states, alphabets, and transitions.
- Automatic NFA to DFA subset construction converter.
- DFA Minimization tool via partition refinement.
- User authentication via Firebase Authentication (Google, GitHub, Microsoft sign-in).
- User-scoped local storage persistence.
- Complete unit and integration test coverage via Vitest.
- Prettier code formatter integration.
