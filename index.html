    First time?

    Build your first DFA in three steps.

    1. 1

      List the states.

      Type them comma-separated in States, e.g. q0, q1. Each state is a snapshot of "where the machine is".

    2. 2

      List the alphabet.

      Type the input symbols in Alphabet, e.g. 0, 1. These are the only symbols your test strings may contain.

    3. 3

      Set start & accept states, then fill the table.

      Pick which state to begin in and which states accept the input. Then for every (state, symbol) cell, type the state to go to.

    or just start typing the fields below.
