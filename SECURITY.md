# Security Policy

## Supported Versions

Currently, security updates are applied to the active `main` branch. We support the following versions:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open a public issue on GitHub. Instead, report it privately to ensure it can be patched before disclosure.

1. Email reports to: **hamza@example.com** (please replace with official project security contact email).
2. Include details of the vulnerability, steps to reproduce, and any potential impacts.
3. We will acknowledge receipt of your report within 48 hours and coordinate a fix release.
